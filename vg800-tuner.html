App Settings

MIDI Output

Choose the MIDI port wired to your VG-800, then set Channel to match the pedal's MIDI · RX CHANNEL (factory default is 1).

The dot on the ⚙ button glows green once a port is live.

Mode

Classic (default) drives the VG-800 preset where ALT TUNE (User mode, saved ON in the patch) carries the tuning, and each bend is the pedal's true two-step gesture: set the target BEND DEPTH, then sweep BEND CONTROL 0→100 so the pitch glides into the note (and glides back on release) instead of stepping. Panning is unavailable in Classic — its CC slots drive the ALT TUNE pitches instead (the VG-800 has only 16 assign slots).

Panning drives the older preset: bends write straight to BEND DEPTH (the note steps to pitch), and CC 71–76 place each string in the stereo field.

Classic mapping: STR BEND(A) · DEPTH 1–6 ← CC 30, 31, 64, 65, 66, 68 (−12/+12) · BEND CONTROL ← CC 69 (0–100) · INST · TYPE ← CC 70 (MIN E.GUITAR / MAX ACOUSTIC) · ALT TUNE(A) · PITCH 1–6 ← CC 71–76 (−24/+24) · INST:E.GTR · TYPE ← CC 77 · INST:ACOUSTIC · TYPE ← CC 78. 12-string has no CC here — toggle it on the pedal (▲) — and ALT TUNE ON is stored in the patch itself.

Guitar model (Auto): each tuning selects its instrument model with two messages — INST TYPE first (electric vs acoustic), then the model on CC 77/78. Bend time now lives in the Bends panel.

Keyboard Mapping

Tap any chip, then press a key to rebind it (Esc cancels). Pedals fire while held (or tap-to-latch). Chords degree keys follow the Key & Scale you pick in the Chords panel; hold Shift with a degree for its 7th. Each section’s ⌨ Map button jumps here.

Clipboard

Rolling always tracks the last N selections. Manual waits for Start, then captures until you Copy. Trim on copy (Rolling only) makes Copy open a draggable timeline so you can copy just part of the log. The Clipboard strip sits under the Current Tuning readout — tunings, chords, instruments and transposes are all logged.

About

Retune your Boss VG-800 from the browser. Pick an alternate tuning and all six strings retune instantly over MIDI — no menu-diving. It also does pedal-steel bends, ethnic-instrument voicings and stereo auto-panning, all from one page.

How to use
Quick start
  • Open in Chrome or Edge (Safari & Firefox lack Web MIDI) and allow MIDI access when asked.
  • Open ⚙ Settings → MIDI Output, pick your interface's output port and set Channel to match the VG-800 (factory default 1). The dot on the ⚙ button glows green when a port is live.
  • Click any tuning card — all six strings retune at once.
  • First time connecting the hardware? Do the one-time VG-800 setup below so it actually responds.
One-time VG-800 setup

The fast way — import a ready-made preset. Download royClassic.tsl (for the default Classic mode) or royPanning.tsl (for Panning mode), open Boss Tone Studio for VG-800 → LIBRARIAN → import the .tsl, and write the patch to the pedal.

Or configure it by hand. The default Classic mode uses all 16 assign slots:

  • INST → ALT TUNE: MODE ALT TUNE, TUNING TYPE USER, switched ON — save it ON in the patch; it carries the tuning (there is no CC for it in this preset).
  • INST → STR BEND: STR BEND SW = ON. The app drives BEND CONTROL itself — resting at 0, swept to 100 for each bend.
  • CONTROL/ASSIGN → ASSIGN (MODE MOMENT everywhere): STR BEND(A) · DEPTH 1–6 ← CC 30, 31, 64, 65, 66, 68 (−12/+12, high-E → low-E) · BEND CONTROL ← CC 69 (0–100) · INST · TYPE ← CC 70 (MIN E.GUITAR / MAX ACOUSTIC) · ALT TUNE(A) · PITCH 1–6 ← CC 71–76 (−24/+24) · INST:E.GTR · TYPE ← CC 77 · INST:ACOUSTIC · TYPE ← CC 78.
  • 12-string has no CC in this preset — toggle it on the pedal itself (▲) when you want it.
  • Panning mode instead? Same DEPTH assigns, but set BEND CONTROL = 100 on the pedal (essential — at 0 the depths are ignored), point CC 71–76 at STRING(A) · PAN 1–6 (MIN L50 / MAX R50), and flip ⚙ Settings → Mode to Panning.
  • Set MIDI · RX CHANNEL to match the app. These CC numbers already line up with Settings → CC numbers out of the box.
Classic vs Panning mode

The ⚙ Settings → Mode toggle picks which of two VG-800 presets the app drives — each spends the pedal's 16 assign slots differently, so each mode is for something different:

  • Classic (default) — for expressive playing: bends that actually glide. The tuning lives on ALT TUNE (User mode, always ON) and every bend is the pedal's true two-step gesture — set the target BEND DEPTH, sweep BEND CONTROL 0→100 — so pedal-steel moves, per-string benders and scale bends slide into pitch like a real mechanical bender instead of stepping. It's also what lets Ethnic instruments sound at true pitch: the ±24 ALT TUNE range absorbs the + Octave lift, so a mandolin plays from the open strings with no capo. The trade-off: no stereo panning — CC 71–76 are spent on the ALT TUNE pitches. Hardware preset: royClassic.tsl.
  • Panning — for the stereo field: per-string placement (Spread, Split, Zig-Zag…) and the auto-pan LFO engine (Rotate, Leslie, Drift…), with CC 71–76 driving STRING · PAN 1–6. Bends still work but write straight to BEND DEPTH, so notes step to pitch instantly rather than glide, and the Ethnic octave rides the preset's fixed +12 on CC 78. Hardware preset: royPanning.tsl.

Pick Classic when the bend gesture matters; pick Panning when the mix does. Match the toggle to whichever preset is loaded on the pedal — the CC numbers overlap, so the wrong mode sends real (wrong) commands.

  • Guitar model auto-select (⚙ Settings → Mode) — in Classic, each tuning can switch the VG-800's instrument model with two messages: INST TYPE (CC 70) picks electric vs acoustic, then CC 77 / CC 78 picks the model. 12-string has no CC in Classic — toggle it on the pedal (▲).
  • Bend time (in the Bends panel) sets how long the Classic glide takes.
Tunings & instruments
  • Tuning cards (Common, Open Majors, Drop, Artists…): click to retune all six strings. Each card shows its note names and per-string ± offset.
  • Ethnic (mandolin, oud, bouzouki, banjo…): maps the instrument's pitches onto a cluster of strings — pick the placement with the string-dot buttons. A capo hint says where to physically capo, since the VG-800 only bends ±12 semitones.
  • Pedal Steel: load a real steel tuning (E9, C6, B6) and choose which 6 of the 10 strings map onto the guitar; the Bends section reconfigures to that tuning's copedent.
  • User: your own tunings, saved on this device. Create dials each string ±12 from standard, names it, and (optionally) saves a pedal/knee-lever setup with it. Export writes a JSON file — your backup and the way to move tunings to another device — and Import reads one back, flagging any name clashes or duplicates.
Chords — key mode, layers & keys

The Chords matrix retunes all six strings to a voiced chord in one tap — it latches, so tap another to change. Pick a Key & Scale and the seven diatonic chords light up, coloured by harmonic function: tonic (green, home), subdominant (blue, departure) and dominant (orange, tension), each with its Roman numeral.

Voicing: Spread spans the neck; Voiced carries a close 3- or 4-string cluster with a bass, played Legato (smooth voice leading between chords) or Grip (identical block shapes).

Layers (the toggle chips under the Voicing row) add harmony without a rainbow — function stays on the fill, extra info rides on badges and borders:

  • 7ths — the diatonic 7th of each degree, marked with a small 7 badge in its function colour.
  • Borrowed — modal-interchange chords, shown as dashed violet cells.
  • Resolve → — hover a dominant chord to ring where it wants to resolve (V7 → I).

Number keys 1–7 play the diatonic degree chords in your current voicing (same latch as the on-screen cells). Modifiers: ⇧+1–7 = that degree's 7th; − / = = 3- / 4-note voicing; [ ] = move the cluster window; 0 = re-center voice leading. Every key here — pedals and chords alike — is rebindable in Settings → Keyboard Mapping (or hit any section's ⌨ Map button).

Scale-tone bends: while a chord is held in key mode, the Bends section becomes six string-pedals (keys A–H). Each raises one string to the next scale tone and springs back on release — a live pedal-steel voice-leading move under a sustained chord (hold String 3 for a 4→3 suspension; hold all six to walk the whole chord up a step).

Diatonic 7th chords by mode — the four-note chord built on each degree. The brass 7 marks the dominant seventh (the leading-tone chord). mM7 = minor-major 7, +M7 = augmented-major 7 — no grid row yet.

Borrowed chords / modal interchange — chords pulled from the parallel mode (same root, opposite major/minor). Flip on the Borrowed layer to surface them. The high-value borrowings in a major key:

  • iv — minor subdominant, the plagal “sigh” (iv → I).
  • ♭VII — the modal / rock cadence ♭VII → I.
  • ♭VI — ♭VI–♭VII–I, the anthem climb.
  • ♭III — a bluesy mediant.
  • ii° — a darker pre-dominant.
Bends — pedals & knee levers
  • Hold the on-screen benders — Sixth, Sus 4, Minor, Maj 7 (keys A S D F) — each named for the colour it pulls the current tuning into, pedal-steel style. They stack, and the icons animate as they move. The Pedal Steel section keeps the authentic Pedal A/B/C names.
  • Combos — IV Chord (the I → IV cry) and Minor 6 engage a whole grip at once and light up the pedals they include. The Minor lever alone turns any major tuning minor; the open-major tunings add IV / V whole-neck pedals that glide the entire chord.
  • Per-String — the Preset ⟷ Per-String switch turns the six pedals into one-per-string manual benders (keys A–H), with factory quick presets (B-Bender, 4th ↑, Nashville…) and savable custom ones. All Strings (J, rebindable) pushes every set bender in one press — e.g. glide the whole guitar up a fourth.
  • Latch — the small Latch button beside ⌨ Map switches pedals to tap on/off instead of holding.
  • Rebind any control in Settings → Keyboard Mapping (or the ⌨ Map button in the header): click a chip, press a key. Reset restores defaults.
  • In Chords key mode the pedals switch to scale-tone bends — see the Chords help above.
  • In Panning mode a bend greys out if base + bend would exceed ±12 (they share one BEND DEPTH parameter). In Classic every bend — pedals, per-string, scale and mode bends alike — rides on top of the ALT TUNE base as its own relative depth, so bends never grey out for range: the full ±12 is available on any tuning.
Panning & auto-pan
  • Available in Panning mode only (⚙ Settings → Mode) — in Classic those CC slots carry the ALT TUNE pitches, so the section hides.
  • Manual modes (Center, Equal Spread, Split, Zig-Zag, Pairs…) place each string in the stereo field. Clicking a mode glides the strings so you feel the sweep; the Pan glide toggle sets the time (Instant → Long).
  • Auto-pan — flip it on and each string runs its own pan LFO. Pick a character (Rotate, Leslie, Fan Breathe, Ping-Pong, Drift), then fine-tune Width, Shape and Phase spread — at a Free rate (Hz) or Tempo-synced (BPM + division).
Current Tuning readout

The always-on panel combines pitch and pan for all six strings: a pan strip up top, then a pitch neck where each note slides flat↔sharp from standard with its offset. Everything updates and animates live from the current tuning, bends and panning. On wide screens it parks in the left sidebar; on very wide screens it moves to a right-hand rail together with the whole Bends panel, so both stay in view while you browse tunings.

Themes & display
  • The pedal button (top-right) opens the theme picker — CS3, DS1, RC500, DD500 and GT1000, each styled after a Boss pedal.
  • Compact mode (Display → Compact) hides the note names and offsets on tuning cards to fit more on screen.
  • Bend effect (Settings) adds a scoop articulation into each new tuning — Panning mode only.
Tips & troubleshooting
  • Nothing moving? Check the green dot (MIDI connected), that the app's Mode (⚙ Settings) matches the preset loaded on the VG-800 — Classic wants ALT TUNE ON (User tuning type) with the 16-slot assign table; Panning wants BEND CONTROL = 100 — and that the app's CC numbers match your assigns.
  • The VG-800 exposes only CC# 0–31 and 64+ and may not track 64–95 continuously — prefer CC# 1–31.
  • Web MIDI needs a secure context: use the hosted page or serve locally over http://localhost (a file:// path won't work).
