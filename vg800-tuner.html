MIDI Settings

Theme

Each string sends its pitch on its own CC. On the VG-800: INST → STR BEND SW = ON, BEND CONTROL = 100. In ASSIGN 1–6 target INST:STR BEND (A) BEND DEPTH 1…6, MIN −12 / MAX +12, MODE = MOMENT, SOURCE = the CC below (DEPTH 1 = high E … DEPTH 6 = low E). Prefer CC# in the 1–31 range — CC# 64–95 are MIDI switch controllers and the VG-800 may not track them continuously.

Volume dip

On the VG-800, map STR BEND · BEND CONTROL (MIN 0 / MAX 100) to Bend Ctrl CC#, and INST · INST LEVEL to Inst Level CC#. When a tuning is applied, both briefly dip then rise back to 100% together with the new pitch. Best with glide off.

CC number per string

Each must match the CC # set as the SOURCE in that string's VG-800 assign slot. Heads-up: the VG-800 only exposes CC# 0–31 and 64+ as assign sources — CC# 32–63 are not selectable.

Panning CC per string

Map to VG-800 INST:STRING(A) PAN 1–6 (MIN L50 / MAX R50). PAN 1 = high E … PAN 6 = low E.
